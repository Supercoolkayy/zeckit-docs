---
title: zeckit down
description: Stop the local devnet and optionally reset state.
---

## Usage

```bash
zeckit down [--purge]
```

## Options

| Option | Description |
| --- | --- |
| `--purge` | Remove local state/volumes and reset to a clean slate |

:::warning
`--purge` is destructive. Use it when you want to fully reset local state.
:::
